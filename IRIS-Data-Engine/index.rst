IRIS Data Engine
======================

IRIS에서 데이터 처리를 위하여 Angora, Jaguar, DSMS 3가지 형태의 데이터 처리 방법을 제공하고 있습니다.

Angora는 대량의 데이터 처리를 위한 엔진으로 Apache Spark을 기반으로 한 데이터 분산 처리에 최적화 되어 있습니다.
데이터 처리를 위한 언어로 자체적으로 정의된 문법을 사용하여 데이터 처리를 진행합니다.

Jaguar는 작은 데이터 처리를 위한 엔진으로 단일 프로세스를 통해 데이터를 처리합니다.
데이터를 가져오기 위해 사용자는 직접 원본 소스에서 제공되는 쿼리를 이용하여 데이터를 검색하고, 데이터 처리를 위해 Jaguar 자체적으로 정의된 문법을 사용하여 데이터 처리를 진행합니다.

DSMS는 RDBMS에 사용자가 작성한 쿼리를 전달하고, 그결과를 반환하여 데이터를 전달 합니다.
데이터 검색 및 처리는 연결된 RDBMS의 기능을 그대로 사용하기 때문에, 해당 RDBMS에서 제공되는 기능을 그대로 사용하여 데이터를 검색할 수 있습니다.
