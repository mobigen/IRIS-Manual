where
====================================================================================================

개요
----------------------------------------------------------------------------------------------------

데이터를 일정 조건에 따라 filter 합니다.

타입
----------------------------------------------------------------------------------------------------
TEXT, INTEGER, BIGINT, REAL, DATE, TIMESTAMP

설명
----------------------------------------------------------------------------------------------------

데이터를 일정 조건에 따라 filter 합니다. 여기서 filter문은 SQL의 WHERE 절과 동일합니다.

Parameters
----------------------------------------------------------------------------------------------------

.. code-block:: none

   ... | where FILTER_CONDITION

.. list-table::
   :header-rows: 1

   * - 이름
     - 설명
     - 필수/옵션
   * - FILTER_CONDITION
     - SQL문에서 WHERE 절에 조건문과 동일 합니다.
     - 필수
