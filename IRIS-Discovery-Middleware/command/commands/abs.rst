abs
======================

개요
-------------------------

절댓값 변환 명령어

컬럼 타입
----------------------------------------------------------------------------------------------------
INTEGER, BIGINT, REAL

설명
--------------------------

integer, bigint, real 등 숫자 컬럼의 절댓값을 구하는 명령어 입니다.

Parameters
------------------------------------

.. code-block:: none
   ... | abs TOKEN
   ... | abs TOKEN, TOKEN


.. list-table::
   :header-rows: 1

   * - 이름
     - 설명
     - 필수/옵션
   * - TOKEN
     - 컬럼을 입력하는 부분 입니다.
     - 필수
